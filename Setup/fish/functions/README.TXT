После добавления не забыть выполнить source ~/.config/fish/functions/файл.fish
codef.fish добавляет команду codef которая открывает vscode файл вместе с его папкой
print.fish добавляет команду print которая используется также как ls из CUPS, требуется наличие CUPS и установка принтера по умолчанию 
bankai.fish добавляет команду bankai которая открывает файл в vim
telegram.fish добавляет команду tg которая открывает телеграмм
red.fish добавляет команду red {ссылка} которая скачивает видео которое идёт после ссылки, поддерживается большинство известных платформ (Требует наличия yt-dlp)
